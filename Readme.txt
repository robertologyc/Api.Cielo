===================================================================
Configuração
===================================================================

Após a instalação é necessário configurar o ambiente, para isso é necessário chamar o método ConfigureEnvironment(), passando como parametros o MerchantId, MerchantKey e o tipo do ambiente (SandBox ou Produção).


Maiores informações acesse https://github.com/robsilva85/Api.Cielo.Lio
